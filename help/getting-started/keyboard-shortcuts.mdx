<Info>
  **Speed up your workflow!** Keyboard shortcuts help you navigate mpath faster and work more efficiently. Most shortcuts work from anywhere in the app.

mpath includes several keyboard shortcuts to help you navigate and work more efficiently. Learn these shortcuts to boost your productivity.

## Essential Shortcuts

These are the shortcuts you'll use most often:

<AccordionGroup>
<Accordion title="⌘K / Ctrl+K - Command Palette">

The **Command Palette** is your fastest way to navigate mpath and perform actions.

**What it does:**
- Opens a searchable menu to navigate to any page
- Search for people, teams, initiatives, or tasks
- Access common actions quickly
- Remembers your recent searches

**How to use:**
1. Press **⌘K** (Mac) or **Ctrl+K** (Windows/Linux)
2. Start typing to search
3. Use arrow keys to navigate results
4. Press **Enter** to select or **Escape** to close

<Info>
  The command palette is context-aware and shows relevant options based on where you are in the app.
</Info>

</Accordion>

<Accordion title="⌘J / Ctrl+J - AI Chat">

Open the **AI Chat** sidebar to get instant answers about your organization.

**What it does:**
- Opens the AI assistant sidebar
- Ask questions about your organization's data
- Get insights about people, teams, and work
- Access help and guidance

**How to use:**
1. Press **⌘J** (Mac) or **Ctrl+J** (Windows/Linux)
2. Type your question in the chat
3. Press **Escape** to close the chat

<Info>
  The AI chat understands your organization's context and can answer questions about people, initiatives, tasks, and more.
</Info>

</Accordion>

<Accordion title="Q - Quick Task Creation">

Create tasks instantly from anywhere in the app.

**What it does:**
- Opens the quick task creation dialog
- Create tasks without navigating to the tasks page
- Perfect for capturing tasks during meetings or conversations

**How to use:**
1. Press **Q** from anywhere in the app
2. Fill in the task details
3. Press **Enter** to save or **Escape** to cancel

<Info>
  Use this shortcut during 1:1s or meetings to quickly capture action items without interrupting the flow.
</Info>

</Accordion>

<Accordion title="E - Quick Edit">

Edit the current page's content instantly.

**What it does:**
- Opens the edit form for the current detail page
- Works on people, initiatives, meetings, tasks, one-on-ones, feedback, and teams pages
- Jump straight into editing mode

**How to use:**
1. Navigate to any detail page
2. Press **E** to open edit mode
3. Make your changes
4. Press **Escape** to cancel editing

<Warning>
  This shortcut only works on detail pages, not list pages. Make sure you're viewing a specific person, initiative, task, etc.
</Warning>

</Accordion>
</AccordionGroup>

## Pro Tips

<AccordionGroup>
<Accordion title="Master the Command Palette">

The command palette is your productivity superpower:

- **Search by name**: Type a person's name, team name, or initiative title to jump directly to it
- **Recent items**: The palette remembers your frequently accessed items
- **Fuzzy search**: You don't need to type exact names - partial matches work too
- **Keyboard navigation**: Use arrow keys to browse, Enter to select, Escape to close

<Info>
  Try typing just a few letters of someone's name or an initiative title - the command palette will find it!
</Info>

</Accordion>

<Accordion title="Workflow Optimization">

Combine shortcuts for maximum efficiency:

1. **During meetings**: Use **Q** to quickly capture action items
2. **Reviewing work**: Use **⌘K** to jump between initiatives and tasks
3. **Getting context**: Use **⌘J** to ask the AI about team members or projects
4. **Quick edits**: Use **E** to make fast updates without clicking around

</Accordion>

<Accordion title="Keyboard-First Workflow">

Try this keyboard-only workflow:

1. Press **⌘K** to open command palette
2. Type to search for what you need
3. Press **Enter** to navigate
4. Press **E** to edit if needed
5. Press **Q** to create related tasks
6. Press **⌘J** to ask questions about context

<Info>
  Once you get comfortable with shortcuts, you'll find yourself rarely needing to use the mouse for navigation!
</Info>

</Accordion>
</AccordionGroup>

## Getting Started

<Steps>
<Step title="Learn the essentials">

Start with these three shortcuts:
- **⌘K / Ctrl+K** - Command palette (your navigation hub)
- **Q** - Quick task creation (capture tasks instantly)
- **⌘J / Ctrl+J** - AI chat (get answers quickly)

</Step>

<Step title="Practice daily">

Use shortcuts in your daily workflow:
- Navigate using the command palette instead of clicking
- Create tasks with **Q** during meetings
- Ask the AI questions with **⌘J** when you need context

</Step>

<Step title="Explore advanced usage">

Once comfortable, try:
- Using **E** to quickly edit pages
- Combining shortcuts for complex workflows
- Discovering shortcuts through the command palette itself

</Step>
</Steps>

## Remember

- **Keyboard shortcuts are optional** - you can always use the mouse if you prefer
- **Shortcuts work consistently** across all pages in mpath
- **Start with one or two** shortcuts and gradually add more
- **The command palette** is your fastest way to navigate - master it first!

<Info>
  The best way to learn shortcuts is to use them! Try replacing one mouse action per day with a keyboard shortcut, and you'll be a power user in no time.
</Info>
