<Info>
  **Welcome to mpath!** This guide will help you get started with the basics.
  You'll learn how to set up your organization, add your team, and start
  managing work effectively.
</Info>

Follow these steps to get started with mpath.

<Steps>
<Step title="Sign up or sign in">

If you're new to mpath, click **Sign Up** and create your account using email or a social provider.

If you already have an account, click **Sign In** to access your organization.

<Warning>
  If you received an invitation email, use that link to join an existing
  organization automatically.
</Warning>

</Step>

<Step title="Create or join an organization">

After signing up, you'll be taken to your dashboard. If you don't have an organization yet, you'll be automatically redirected to set one up.

**Creating a new organization:**

1. Enter your organization name (e.g., "Acme Corp")
2. Choose a unique slug (e.g., "acme-corp") - this will be part of your organization URL
3. Click **Create Organization**

**Joining an existing organization:**

- If you have pending invitations, they'll appear on this page - click to accept
- Or ask an organization admin to invite you using your email address

</Step>

<Step title="Add people to your organization">

1. Navigate to **People** from the main navigation
2. Click **Add Person** or **Invite Person**
3. Enter their name and email address
4. Optionally assign them to a team and set their manager

<Accordion title="Quick tip: Build your team structure">
  
Set up your organizational hierarchy by:
- Creating teams under **Teams**
- Assigning people to teams
- Setting manager-report relationships

This helps organize work and enables features like 1:1s and feedback campaigns.

</Accordion>

</Step>

<Step title="Create your first initiative">

Initiatives help you track multi-week projects and strategic work:

1. Go to **Initiatives** from the navigation
2. Click **New Initiative**
3. Give it a name and description
4. Add objectives and key results to track progress
5. Set a status (Planned, In Progress, On Hold, Complete)

<Info>
  Initiatives are your top-level work containers. You'll link tasks, meetings,
  and documents to them for a single source of truth.
</Info>

</Step>

<Step title="Create and assign tasks">

Break down your initiative into actionable tasks:

1. From an initiative page, click **Add Task**
2. Or go to **Tasks** to create standalone tasks
3. Fill in the task details:
   - Title and description
   - Assign to a person
   - Set priority and due date
   - Link to an initiative (optional)
4. Save the task

Tasks can be viewed in the **Tasks** page or within their linked initiative.

</Step>

<Step title="Schedule your first 1:1">

One-on-ones are private meetings between managers and their reports:

1. Go to **One-on-Ones** from the navigation
2. Click **New One-on-One**
3. Select the manager and report
4. Set the frequency (weekly, bi-weekly, monthly)
5. Add an initial agenda or notes

<Info>
  Only the manager and report can see the details of their 1:1 meetings. This
  ensures privacy for sensitive conversations.
</Info>

</Step>
</Steps>

## Essential Navigation Tips

<AccordionGroup>
<Accordion title="Command Palette (⌘/Ctrl + K)">

Press **⌘/Ctrl + K** (or **⌘/Ctrl + P** on Windows) to open the command palette. This lets you:

- Jump to any page quickly
- Create new items (tasks, initiatives, people)
- Search across the application

</Accordion>

<Accordion title="Help icons">

Look for the small **help icon** (?) on pages. Click it to see contextual help and learn more about that feature.

</Accordion>

<Accordion title="Dashboard overview">

Your **Dashboard** shows:

- Tasks assigned to you
- Upcoming 1:1s
- Active feedback campaigns
- Your direct reports
- Recent activity

Use it as your daily starting point.

</Accordion>
</AccordionGroup>

## Common First Steps

Once you're set up, here are typical next actions:

1. **Build your team structure** - Add all team members and set up reporting relationships
2. **Create initiatives** - Define your current strategic projects
3. **Break down work** - Create tasks and link them to initiatives
4. **Schedule recurring 1:1s** - Set up regular check-ins with your team
5. **Run a feedback campaign** - Collect structured feedback for performance reviews or growth cycles
6. **Connect integrations** - Link GitHub or Jira accounts to surface work activity
